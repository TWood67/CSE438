************************************************************
		Part One Instructions			
************************************************************
Navigate to Lab_03/Part1. From here you can execute main
(./main). Instructions will display in the screen with four
options: read, write, set, and quit. Enter the value
associated with the menu depending on which operation is to
be done. For example, entering 1 will allow you to read.
Further instructions will display to guide you as to how to
operate the program.

************************************************************
		Part Two and Three Instructions			
************************************************************
Navigate to Lab_03/Part2 or Lab_03/Part3. Insert the 
i2c_flash module (insmod i2c_flash.ko). Run the executable
(./main). The main instructions are similar to Part 1. When 
finished, remove the device from kernel (rmmod i2c_flash.ko).
